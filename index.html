Step-by-Step Guide In-Class

Step 1: Prepare Your Image

1
Open Your Drawing or Collage
  1. Open your image in Photoshop
  2. Cut out the part you want to use
  3. Copy and paste it onto a new layer
  4. Delete the background

Step 2: Increase Contrast

2
Adjust Levels
  1. Go to Image ‚Üí Adjustments ‚Üí Levels
  2. Move the sliders to make the image clearer

Step 3: Clean Up the Image

3
Remove Unwanted Areas
  1. Use the Elliptical Marquee Tool to select white areas
  2. Go to Select ‚Üí Similar, then press Delete
  3. Press Cmd + D (Mac) / Ctrl + D (Windows) to deselect
  4. Use the Crop Tool to crop tightly around your image
  5. Click the ‚úì (tick) to apply

Step 4: Create the Pattern

4
Define Your Pattern
  1. Make sure the correct layer is selected
  2. Go to Edit ‚Üí Define Pattern
  3. Give your pattern a clear name
  4. Click OK

‚úÖ Done!

Your pattern is now saved.

Step 5: Apply the Pattern

5
Use Your Pattern
  1. Open a design page (or a blank page to test)
  2. Use the Pen Tool to draw a path
  3. Create a new layer
  4. Go to Edit ‚Üí Fill
  5. Choose Pattern and select your pattern
  6. Click OK

Step 6: Experiment

6
Try Different Options
  • Change the size and spacing
  • Try different paths (straight, curved)
  • To remove the blue path line: open the Paths panel and delete the Work Path

üí° Take Your Time

Play and experiment. Good results come from trying different options.
